# Role Radar — Job-Profile Matching Dataset

Training data for a local-first job-candidate fit scoring model. English-language, India-focused.

## Dataset Structure

### Jobs (`scraped_jobs.json`)

2,500 real job postings from Indian cities with full descriptions.

| Field | Description |
|-------|-------------|
| `id` | Unique job ID |
| `title` | Job title |
| `company` | Company name |
| `location` | City / Remote |
| `description` | Full job description text |
| `seniority_level` | Entry/Associate/Mid-Senior/Director/Executive |
| `industry` | LinkedIn industry classification |
| `role_family_hint` | Inferred role family |

**Coverage:** 12 cities, 15 role families (fullstack, backend, frontend, devops, data, mobile, product, sales, marketing, hr, finance, operations, customer_success, legal, systems)

### Profiles (`synthetic_profiles.json`)

640 candidate profiles (438 synthetic, 202 extracted from public resumes).

| Field | Description |
|-------|-------------|
| `profile_id` | Unique profile ID |
| `roles` | Job titles held |
| `skills_primary` | 3-6 primary skills |
| `skills_secondary` | 1-3 secondary skills |
| `experience_years` | Years of experience |
| `seniority` | Junior/Mid/Senior/Staff |
| `domains` | Industry domains |
| `preferences` | Location, remote, salary prefs |

### Pairs & Labels

22,465 profile-job pairs labeled with dimension scores:

| Field | Description |
|-------|-------------|
| `pair_id` | `{profile_id}_{job_id}` |
| `skills` | 50-100 skills fit score |
| `seniority` | 50-100 seniority match |
| `domain` | 50-100 domain match |
| `location` | 50-100 location match |
| `composite` | Weighted sum |

Labels generated by Gemini 2.5 Flash Lite (skills), deterministic scorers (seniority, domain, location).

### Gold Eval (`gold_labels.json`)

108 human-reviewed evaluation pairs with balanced role family coverage.
